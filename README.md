# NextXO Employee Monitor

## Project Info

**NextXO Employee Monitor** is an advanced AI-powered workplace analytics and compliance tracking system. It features real-time monitoring, incident detection, and custom AI model training capabilities.

## Features

- **Real-time Dashboard**: Monitor employee activities and compliance status.
- **AI Training**: Train custom models to detect specific objects or behaviors directly in the browser.
- **Dataset Management**: Annotate images and manage datasets for training.
- **Incident Reporting**: Automatic logging of compliance incidents.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd employee-monitor-view
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

5. Start the backend server (for dataset saving):
   ```sh
   node server.js
   ```
